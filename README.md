## Network Port Scanner (Python)

## Project Overview
This project is a simple **Network Port Scanner built using Python**. 
The tool scans a target system to identify **open TCP ports** within a specified range.

Port scanning is a common technique in cybersecurity and penetration testing to identify running services on a network host.

---

## Features
- Accepts **target IP address** as input
- Scans **ports 20–100**
- Identifies **open TCP ports**
- Simple and beginner-friendly Python implementation

---

## Technologies Used
- Python
- Socket Library

---

## How It Works
1. The user enters a **target IP address**.
2. The script creates a **TCP socket connection**.
3. It scans ports **20 to 100**.
4. If a connection is successful, the port is marked as **OPEN**.
